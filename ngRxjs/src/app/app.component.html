Angular RxJS - Reactive Extension for JavaScript

What is RxJS?

Ans: RxJS is a JavaScript implementation of Reactive Extension API. RxJS provides single API to deal with the data that is coming from multiple sourse.


What is Stream?

Ans: The data that is coming continuosly coming to your application is called as data stream.

Stream of data can be -

  • Synchronous Data
  • Asynchronous Data
  • Getting Data from the server as HTTP calls
  • DOM events
  • Function with return value
